This plug-in requires u3a-siteworks-core to be activated. It enables Group finances to be entered for each Group recorded under Groups.
The shortcode ['u3a_finance_form' '2023'] where 2023 is the year for which the finances are relevant on a suitable page. This page can either be password protected or require a log-in if you have that facility.

The details are stored in a csv file either from the Group finances section of the dashboard or by putting the code: <a href=../../wp-content/uploads/groupfinances/groupfinances.csv download>Download the finance file</a> on a suitable page.

The treasurer is set as an 'option'. It gets updated from the admin page.
